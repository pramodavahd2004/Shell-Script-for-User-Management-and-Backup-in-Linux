# Shell Script for User Management and Backup in Linux

## Project Overview
This project is a Linux Bash shell script that automates user and group management tasks and performs secure backups of directories. It provides a menu-driven command-line interface to simplify system administration and improve efficiency.

## Objective
To develop a shell script that:
- Automates Linux user and group management
- Creates compressed and timestamped backups of directories
- Enhances system security and reliability
- Uses Git and GitHub for version control and project sharing

## Features
- Add, delete, and modify user accounts  
- Create and manage groups  
- Add users to groups  
- Perform compressed directory backups using `tar`  
- Menu-driven, user-friendly interface  
- Timestamped backup files for easy identification  

## Tools & Technologies
- Linux (Ubuntu / CentOS / Fedora)
- Bash Shell Scripting
- Git & GitHub
- Linux Commands: useradd, userdel, usermod, groupadd, usermod -aG, tar
